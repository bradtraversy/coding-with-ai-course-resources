# Implement Tagging

Now we want the user to be able to tag items so that they have even faster, easier access.

Let's run the following prompt:

```text
/feature load Add tagging system. Users should be able to add tags when then create/edit and item. The tags should be shown on in the drawer view and we should be able to filter by tag in the search/filter input
```

Run `/feature start`

Check the cards and drawers and make sure you can filter by tag.

```text
/feature test
/feature review
/feature complete
/clear
```
