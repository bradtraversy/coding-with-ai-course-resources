# Markdown Editor

For our snippets and commands, we used a code editor. For notes and prompts, we want to use a markdown editor.

Run the following command:

```text
/feature load Add a markdown editor and apply it to the note and prompt types. The editor should  have a preview area in a tab to easily switch back and forth.
```

Now run it with `/feature start`.

Let's add a language dropdown to the modal as well:

```text
For the add/edit modal when using a snippet type, create a language dropdown and reflect the highlighting and formatting for the selected language. For commands, it should always be bash.
```

Iterate if needed. Make sure that the preview works and is easily accessed.

Once finished, run:

```text
/feature test
/feature review
/feature complete
/clear
```
