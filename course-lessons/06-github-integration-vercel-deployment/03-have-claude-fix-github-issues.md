# Have Claude Fix Github Issues

Since we setup the CLI and Claude has access to it, we can now do things like have it fix issues.

Go to the repo on Github and create a new issue. Just something simple.

Now in your Claude prompt, ask to list the issues:

```text
Show me the open issues in this repository on Github
```

Ask to fix the specific issue and then close it.

```text
Fix the X issue and then mark it as closed
```

It's as simple as that. You could even setup automation to have them fixed automatically.
